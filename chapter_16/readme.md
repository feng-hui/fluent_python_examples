#### 1、什么是协程？

协程是指一个过程，这个过程与调用方协作，产出由调用方提供的值。

**协程的使用和特性**

* 生成器的调用方可以使用`.send(...)`方法发送数据，`.throw()`方法让调用方抛出异常，`.close()`贩方法是终止生成器。
* Python3.3（2012年）对生成器的语法新增了两处改动：
    * 生成器可以返回一个值，Python3.3之前的版本是不可以的，如果给return提供值，会抛出SyntaxError异常；
