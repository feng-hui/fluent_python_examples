1、什么是函数装饰器？

用于在源码中“标记”函数，以某种方式增强函数的行为。

```
@decorator
def target():
    print('running target()')
```

等同于

```
def target():
    print('running target()')

target = decotrator(target)
```

2、Python如何执行装饰器？

* 装饰器的一个关键特性是，他们在导入的时候立即运行。

* 装饰器通常是在一个模块中定义，然后应用到其他模块的函数上。


3、闭包
